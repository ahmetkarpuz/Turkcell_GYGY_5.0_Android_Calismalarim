**GNCYTNK Bankası - Temel Bankacılık Uygulaması**
Bu proje, Kotlin dilinin temel yapılarını (fonksiyonlar, değişkenler, döngüler ve karar yapıları) kullanarak geliştirilmiş basit bir konsol tabanlı bankacılık uygulamasıdır. Nesne Yönelimli Programlama (OOP) kullanılmadan yapılmıştır.

**Özellikler:** 
Bakiye Sorgulama,
Para Yatırma,
Para Çekme,
Hesap Detayları,
Para Transferi
